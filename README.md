# Sistema de Cadastro com Flask

Projeto desenvolvido com Python utilizando Flask para simular um sistema de cadastro de usuários.

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Interface web com HTML e CSS

## Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
